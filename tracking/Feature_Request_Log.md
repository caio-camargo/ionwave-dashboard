## REQUEST STATUS

- **Submitted:** Request received, needs review

## PRIORITIZATION CRITERIA

1. **Customer Impact:** How many customers want this?
2. **Business Value:** Does it drive revenue/retention?
3. **Effort:** Development time required
4. **Strategic Fit:** Aligns with product vision?
5. **Urgency:** Time-sensitive?
