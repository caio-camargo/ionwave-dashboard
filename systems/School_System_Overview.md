# School System Overview

**Purpose:** Training and development system for Studio 3 participants  
**Applies To:** Operators, CMs, GMs, VPs  
**Owner:** Studio 3 System

---

## WHAT IS THE SCHOOL

**The School is Studio 3's internal training system.**

Instead of hiring only experienced people, School enables:
- Training new Operators
- Upskilling existing team
- Standardizing approaches
- Building institutional knowledge

**Metaphor:** Like a business school, but for Studio 3 methodology

---

## SCHOOL STRUCTURE

### Semester System
**Duration:** 4 months per semester  
**Cadence:** 3 semesters per year  
**Why:** Long enough for real progress, short enough to maintain urgency

### Programs
**6 core programs:**
1. Operator Program
2. CM Program
3. GM Program
4. VP Program
5. Marketing Program
6. Operations Program

**Each program:** Curriculum, milestones, graduation criteria

---

## SEMESTER ARCHITECTURE

### Month 1: Foundation
- Core concepts
- Tool training
- Process familiarization
- Build baseline

### Months 2-3: Application
- Apply to real Trade (or simulation)
- Supervised execution
- Weekly reviews
- Coaching and feedback

### Month 4: Mastery
- Independent work
- Final assessments
- Graduation evaluation
- Next semester planning

## ENROLLMENT MATRIX

|------|-----------|-----------|-----------|

## GRADUATION CRITERIA

**To graduate program:**
- All deliverables submitted
- Average grade B or higher
- Pass final assessment
- Supervisor recommendation

**Upon graduation:**
- Certificate of completion
- Progress to next semester
- OR graduate to full role

## SCHOOL MODEL OVERVIEW

**The School is NOT:**
- Academic education
- Theory without practice
- Classroom lectures
- Divorced from real work

**The School IS:**
- Practical training
- Learn by doing
- Real Trade work
- Systematic skill building

**Delivery:** Combination of:
- Async materials (read/watch)
- Sync coaching (weekly)
- Real execution (on Trades)
- Peer learning (cohorts)
